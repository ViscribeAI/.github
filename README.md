# ViscribeAI

**Viscribe is the Image Intelligence Layer for AI Agents** — AI-powered image understanding and analysis to describe, classify, extract, compare, and answer questions about images.

---

## Repositories

| Repository | Description |
|------------|-------------|
| [**python-sdk**](https://github.com/ViscribeAI/python-sdk) | Official Python SDK for the Viscribe API — sync/async clients, Pydantic schemas, and full image endpoints. |
| [**langchain-viscribe**](https://github.com/ViscribeAI/langchain-viscribe) | LangChain integration for Viscribe — use Viscribe as tools and components in your LangChain agents. |
| [**viscribe-mcp-server**](https://github.com/ViscribeAI/viscribe-mcp-server) | MCP (Model Context Protocol) server for Viscribe — bring image intelligence to any MCP-compatible AI agent. |
